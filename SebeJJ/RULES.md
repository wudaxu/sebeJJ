# SebeJJ 项目管理规则

## 自动处理规则

### Agent 中断自动恢复
- 美术 Agent 中断 → 自动重启，继续当前周任务
- 程序 Agent 中断 → 自动重启，继续当前周任务
- 测试 Agent 中断 → 自动重启，继续当前周任务
- 架构 Agent 中断 → 自动重启，继续当前周任务

### 中断检测
- 每 5 分钟检查一次 Agent 状态
- 发现 `abortedLastRun: true` 立即重启
- 重启时携带上下文，确保任务连续性

### 进度汇报
- 用户询问时立即汇报
- 每完成一个 Week 自动汇总
- 重大问题立即通知用户

## 项目时间线
- Week 1: ✅ 已完成（基础架构）
- Week 2: 🔄 进行中（机甲/采集/音效）
- Week 3: ⏳ 待开始（下潜/场景）
- Week 4: ⏳ 待开始（整合/Polish）
