# SebeJJ 赛博朋克风格指南

## 项目概述
**SebeJJ** - 赛博机甲深海资源猎人

## 色彩系统

### 主色调
- **赛博青 (Cyber Cyan)** - `#00FFFF` - 主强调色，能量、科技
- **霓虹粉 (Neon Pink)** - `#FF00FF` - 危险、警告、敌人
- **电光紫 (Electric Purple)** - `#8B00FF` - 稀有物品、特殊事件

### 辅助色
- **深海蓝 (Deep Blue)** - `#001133` - 背景主色
- **暗海蓝 (Dark Ocean)** - `#002244` - 次级背景
- **生物荧光绿 (Bio-Green)** - `#00FF88` - 生命、氧气、友好
- **警示橙 (Alert Orange)** - `#FF8800` - 警告、低状态
- **危险红 (Danger Red)** - `#FF0044` - 危险、伤害

### 中性色
- **钛白 (Titanium White)** - `#E0E0E0` - 文字、高亮
- **钢灰 (Steel Gray)** - `#8899AA` - 次要文字
- **墨黑 (Ink Black)** - `#0A0A0F` - 阴影、边框

## 视觉风格

### 赛博朋克元素
1. **霓虹光效** - 所有UI元素带发光效果
2. **全息投影** - 半透明叠加层，扫描线效果
3. **故障艺术** - 偶尔的glitch效果
4. **网格/线条** - 几何装饰线条
5. **数据可视化** - 数字、图表、波形

### 深海元素
1. **渐变深度** - 从浅蓝到深黑的垂直渐变
2. **粒子效果** - 海洋雪、气泡、浮游生物
3. **生物发光** - 有机光源
4. **压强视觉** - 越深越暗，对比度降低

## 字体规范

### 标题字体
- **Orbitron** - 科技感标题
- **Rajdhani** - 次级标题

### 正文字体
- **Exo 2** - UI文字
- **Share Tech Mono** - 数据/数字显示

## UI组件规范

### 按钮
- 边框: 2px solid #00FFFF
- 背景: transparent → rgba(0,255,255,0.1) hover
- 文字: #00FFFF
- 圆角: 4px
- 发光: box-shadow: 0 0 10px rgba(0,255,255,0.5)

### 状态条
- 边框: 1px solid #445566
- 背景: rgba(0,0,0,0.5)
- 填充: 渐变色根据类型
- 高度: 20px

### 面板
- 背景: rgba(0,17,51,0.85)
- 边框: 1px solid #00FFFF
- 内发光: inset 0 0 20px rgba(0,255,255,0.1)
- 装饰角: 4个角的三角形装饰

## 机甲设计规范

### Mk.I 机甲
- **风格**: 工业风 + 赛博朋克
- **主色**: 深灰 + 青色发光
- **特征**: 
  - 球形驾驶舱
  - 机械臂 x2
  - 推进器 x4
  - 货舱模块
  - 传感器阵列

### 发光部件
- 驾驶舱窗口: 青色
- 推进器: 青色/蓝色
- 传感器: 绿色脉冲
- 武器接口: 红色待命

## 特效规范

### 粒子效果
- 气泡: 白色半透明，向上漂浮
- 火花: 橙色/黄色，短寿命
- 能量: 青色，持续发光
- 血迹/液体: 深红色，向下飘散

### 光照
- 机甲探照灯: 锥形白光
- 生物发光: 柔和彩色点光源
- 爆炸: 强光闪烁

## 文件命名规范

```
[类型]_[名称]_[变体]_[版本].[格式]

类型:
- mech_ 机甲
- ui_ 界面
- bg_ 背景
- fx_ 特效
- icon_ 图标

示例:
- mech_mk1_base_v1.svg
- ui_hud_healthbar_v1.svg
- bg_deepsea_1000m_v1.svg
```
