## 概述

### 设计理念

深度区域系统是 SebeJJ 的核心玩法之一，玩家驾驶机甲探索不断加深的海底世界。每个深度区域都有独特的环境挑战、敌人、资源和特殊机制。

### 核心机制

- **压强系统**: 深度越深，压强伤害越高
- **能见度**: 影响玩家视野和探索体验
- **渐进解锁**: 必须按顺序解锁区域
- **安全区检查点**: 提供补给和存档点

### 区域概览表

| 深度 | 区域名称 | 能见度 | 压强伤害 | 难度等级 |
|------|----------|--------|----------|----------|
| 200m | 暗礁区 | 60% | 2/秒 | ★★☆☆☆ |
| 500m | 沉船区 | 40% | 3/秒 | ★★★☆☆ |
| 1000m | 深渊入口 | 20% | 5/秒 | ★★★★☆ |

## 配置文件说明

### 文件结构

```
/root/.openclaw/workspace/projects/sebejj/Assets/Resources/DepthAreas/
├── Area_200m_ReefZone.json      # 200米暗礁区配置
├── Area_500m_ShipwreckZone.json  # 500米沉船区配置
├── Area_1000m_AbyssEntrance.json # 1000米深渊入口配置
└── AreaUnlockSystem.json         # 区域解锁系统配置
```

### 配置格式

所有配置文件采用JSON格式，包含以下主要部分:

1. **基础信息**: areaId, areaName, depth
2. **场景配置**: visibility, pressureDamage, ambientColor等
3. **敌人配置**: enemies数组
4. **资源配置**: resources数组
5. **特殊机制**: specialMechanics
6. **解锁条件**: unlockRequirements
7. **检查点**: checkpoint

### 使用方式

```csharp
// 加载区域配置
var areaConfig = JsonUtility.FromJson<DepthAreaConfig>(
    Resources.Load<TextAsset>("DepthAreas/Area_200m_ReefZone").text
);

// 应用场景配置
RenderSettings.ambientLight = areaConfig.sceneConfig.GetAmbientColor();
FogManager.SetDensity(areaConfig.sceneConfig.fogDensity);
```

### 压强伤害公式

```
实际伤害 = 基础压强伤害 × 深度系数 × (1 - 抗性减免)

其中:
- 深度系数 = 当前深度 / 1000
- 抗性减免 = 抗压服抗性% + 技能抗性%
```
