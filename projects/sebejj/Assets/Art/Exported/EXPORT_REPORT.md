```yaml
Pixels Per Unit: 100
Filter Mode: Point (no filter)
Compression: None (for pixel art)
```

### UI元素
```yaml
Texture Type: Sprite (2D and UI)
Sprite Mode: Single
Pixels Per Unit: 100
Filter Mode: Bilinear
Compression: Normal Quality
```

```yaml
Sprite Mode: Single
Pixels Per Unit: 100
Filter Mode: Point
```
