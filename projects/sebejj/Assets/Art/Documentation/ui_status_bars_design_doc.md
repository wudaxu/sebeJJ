# UI状态条 (Status Bars) 美术资源设计说明

## 资源清单

### 状态条
- `ui_health_bar.svg` - 生命值条
- `ui_energy_bar.svg` - 能量条
- `ui_oxygen_bar.svg` - 氧气条

## 设计规范

### 通用结构 (9-patch)
```
+--------+--------+--------+
| 固定   | 拉伸   | 固定   |
| 4x4px  | 水平   | 4x4px  |
+--------+--------+--------+
| 拉伸   | 填充   | 拉伸   |
| 垂直   | 区域   | 垂直   |
+--------+--------+--------+
| 固定   | 拉伸   | 固定   |
| 4x4px  | 水平   | 4x4px  |
+--------+--------+--------+
```

### 配色方案

#### 生命值条 (Health)
- **边框**: 深红 (#FF0040)
- **填充渐变**: 深红 → 粉红 → 浅粉
- **低血量**: 红色 → 橙色警告
- **高光**: 白色 (20%透明度)

#### 能量条 (Energy)
- **边框**: 赛博青 (#00F0FF)
- **填充渐变**: 蓝 → 青 → 绿
- **动态效果**: 流动条纹动画
- **高光**: 白色 (25%透明度)

#### 氧气条 (Oxygen)
- **边框**: 青蓝 (#00D4FF)
- **填充渐变**: 青绿 → 青蓝
- **纹理**: 气泡图案
- **低氧**: 橙色 → 红色警告
- **高光**: 白色 (25%透明度)

### 视觉特征
1. **四角装饰**: 45度切角线条
2. **刻度线**: 25%、50%、75%位置标记
3. **发光边框**: 霓虹光晕效果
4. **高光反射**: 上半部分光泽

## PNG导出规格

### 基础尺寸
| 参数 | 规格 |
|------|------|
| 分辨率 | 128x32 px |
| 格式 | PNG-24 (透明通道) |
| 9-patch标记 | 1px黑色边框 |

### 9-patch导出
```
文件名格式:
- ui_health_bar.9.png
- ui_energy_bar.9.png  
- ui_oxygen_bar.9.png

标记位置:
- 顶部: 16px开始，96px宽度可拉伸
- 左侧: 8px开始，16px高度可拉伸
```

### 缩放版本
| 尺寸 | 用途 |
|------|------|
| 128x32 | 默认 |
| 256x64 | 高清/大屏 |
| 64x16 | 小图标/紧凑模式 |

## 使用建议

### Unity 9-slice设置
```
Texture Type: Sprite (2D and UI)
Sprite Mode: Single
Mesh Type: Full Rect
Pixels Per Unit: 100
Border: L:16 R:16 T:8 B:8
```

### 代码动态填充
```csharp
// 填充比例 0-1
healthBar.fillAmount = currentHealth / maxHealth;

// 低血量闪烁
if (healthPercent < 0.2f) {
    healthBar.color = Color.Lerp(normalColor, warningColor, flashSpeed);
}
```
