# 战斗特效 (Combat Effects) 美术资源设计说明

## 资源清单

### 基础特效
- `fx_hit_spark.svg` - 击中火花
- `fx_explosion_small.svg` - 小型爆炸
- `fx_shield_impact.svg` - 护盾受击

### 武器特效
- `fx_drill_spark.svg` - 钻头火花
- `fx_laser_beam.svg` - 激光束

## 设计规范

### 击中火花 (Hit Spark)
- **用途**: 近战武器击中敌人
- **配色**: 白 → 黄 → 橙 → 红
- **结构**: 中心闪光 + 4主火花 + 4次火花 + 碎片
- **动画**: 4-6帧快速播放 (约0.1秒)

### 小型爆炸 (Small Explosion)
- **用途**: 爆炸武器、地雷
- **配色**: 白 → 黄 → 橙 → 红 → 黑烟
- **结构**: 外圈烟雾 + 爆炸主体 + 飞散碎片
- **动画**: 6-8帧，由小变大再消散

### 护盾受击 (Shield Impact)
- **用途**: 护盾被击中
- **配色**: 电光紫 + 赛博青
- **结构**: 护盾网格 + 涟漪扩散 + 能量飞溅
- **动画**: 4-6帧，涟漪向外扩散

### 钻头火花 (Drill Spark)
- **用途**: 钻头攻击持续效果
- **配色**: 白 → 黄 → 橙
- **结构**: 中心火花 + 飞溅火花 + 金属碎屑
- **动画**: 4-6帧快速循环

### 激光束 (Laser Beam)
- **用途**: 激光武器发射
- **配色**: 白芯 + 品红外圈
- **结构**: 核心光束 + 外圈光晕 + 能量脉冲
- **动画**: 2-4帧闪烁循环

## PNG导出规格

### 通用规格
| 参数 | 规格 |
|------|------|
| 格式 | PNG-24 (透明通道) |
| 色彩空间 | sRGB |
| 压缩 | 无损 |

### 各特效尺寸

| 特效 | 尺寸 | 帧数 | 播放方式 |
|------|------|------|----------|
| fx_hit_spark | 128x128 | 4-6 | 单次 |
| fx_explosion_small | 128x128 | 6-8 | 单次 |
| fx_shield_impact | 128x128 | 4-6 | 单次 |
| fx_drill_spark | 128x128 | 4-6 | 循环 |
| fx_laser_beam | 256x64 | 2-4 | 循环 |

### 导出设置
```
- 抗锯齿: 开启
- 透明背景: 是
- 嵌入色彩配置文件: 否
```

## 使用建议

### Unity粒子系统设置
```
Material: Sprites-Default 或 Additive
Sorting Layer: Effects
Order in Layer: 10
```

### 动画播放参数
```csharp
// 击中火花
hitSpark.Play("fx_hit_spark", 0, 0);
hitSpark.speed = 1.5f; // 加速播放

// 激光束
laserBeam.loop = true;
laserBeam.Play("fx_laser_beam");
```

### 层级管理
- 特效应位于角色上层
- 多个特效使用对象池管理
- 播放完成后自动回收
