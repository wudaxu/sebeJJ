# 武器图标 (Weapon Icons) 美术资源设计说明

## 资源清单

### 武器图标
- `weapon_icon_drill.svg` - 钻头
- `weapon_icon_claw.svg` - 机械爪
- `weapon_icon_laser.svg` - 激光

## 设计规范

### 通用结构
- **尺寸**: 64x64 px
- **背景**: 深海黑 (#0D1B2A) 圆角矩形
- **边框**: 2px 霓虹色
- **内部留白**: 4px

### 配色方案

#### 钻头 (Drill)
- **边框**: 赛博青 (#00F0FF)
- **主体**: 深灰蓝 + 金属色
- **特效**: 钻头火花 (橙黄)
- **特征**: 锥形钻头 + 螺纹线条

#### 机械爪 (Claw)
- **边框**: 赛博青 (#00F0FF)
- **主体**: 深灰蓝 + 关节发光
- **特征**: 双爪设计 + 液压杆

#### 激光 (Laser)
- **边框**: 霓虹粉 (#FF00A0)
- **主体**: 深灰蓝 + 品红核心
- **特效**: 枪口发光
- **特征**: 枪管 + 能量条

## PNG导出规格

| 参数 | 规格 |
|------|------|
| 分辨率 | 64x64 px |
| 格式 | PNG-24 (透明通道) |
| 色彩空间 | sRGB |
| 文件名格式 | weapon_icon_[武器名].png |

### 缩放版本
| 尺寸 | 用途 |
|------|------|
| 64x64 | 默认 |
| 128x128 | 高清 |
| 32x32 | 小图标 |

## 使用建议

### UI布局
- 武器槽位: 64x64 或 48x48
- 快捷栏: 32x32
- 装备界面: 128x128

### Unity导入设置
```
Texture Type: Sprite (2D and UI)
Sprite Mode: Single
Pixels Per Unit: 100
Filter Mode: Bilinear
Compression: None
```
