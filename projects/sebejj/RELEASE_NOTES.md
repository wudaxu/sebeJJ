## ✨ 主要功能

### 1. 委托系统 v1.0
- **15个初始委托** (Q001-Q015设计，Q001-Q008已实现)
- **5种委托类型**: 采集、歼灭、探索、护送、狩猎
- **动态难度系统**: 5个难度等级 (新手级到深渊级)
- **完整的新手教程**: Q001新手试潜引导

### 2. 机甲系统 v1.0
- **3种机甲框架**: 轻型"幽灵"/标准"守护者"/重型"堡垒"
- **8种装备类型**: 钻头、采集臂、扫描仪、推进器等
- **6种功能模块**: 货舱扩展、能源核心、装甲板等
- **完整的属性系统**: 船体、能量、速度、货舱

### 3. 战斗系统 v1.0
- **完整伤害计算**: 含护甲、护盾、防御计算
- **3种武器**: 链锯(近战)、等离子炮(远程)、EMP(控制)
- **连击系统**: 支持武器连击和伤害递增
- **击杀反馈**: 视觉和音频反馈

### 4. AI系统 v1.0
- **状态机框架**: 完整的AI状态机实现
- **A*寻路系统**: 2D网格寻路，支持动态障碍物
- **3种敌人AI**:
  - 机械鱼: 游荡、追击、冲撞攻击
  - 机械蟹: 巡逻、防御、钳击连击
  - 机械水母: 漂浮、电击AOE
- **Boss AI**: 铁钳巨兽完整实现

### 5. UI系统 v1.0
- **完整的动画框架**: 16种UI动画效果
- **血条系统**: 平滑血量变化动画
- **伤害数字**: 浮动伤害显示
- **任务提示**: 任务完成和进度提示

### 6. 体验优化 v1.0
- **教程系统**: 分步骤新手引导
- **难度曲线**: 动态难度调整
- **死亡惩罚**: 合理的惩罚机制
- **节奏控制**: 战斗和探索节奏管理

## 🚀 安装和运行

### 系统要求
- **Unity**: 2022.3 LTS 或更高版本
- **平台**: Windows 10/11, macOS 12+, Linux
- **内存**: 最低 4GB，推荐 8GB
- **显卡**: 支持 DirectX 11

### 安装步骤

1. **克隆项目**
```bash
git clone <repository-url>
cd sebejj
```

2. **Unity Hub 打开**
   - 打开 Unity Hub
   - 点击 "Open" → 选择项目文件夹
   - 使用 Unity 2022.3 LTS 打开

3. **首次导入**
   - 等待 Unity 导入所有资源
   - 打开 `Assets/Scenes/MainScene.unity`

4. **运行游戏**
   - 点击 Play 按钮
   - 开始 Q001 新手试潜
