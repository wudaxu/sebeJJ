- 背包管理系统

  - 1000-1500米: 深海平原
  - 1500-2000米: 深渊核心
  - 2000米+: 远古遗迹
- 环境效果系统
- 氧气管理
- 深度压力机制

#### 战斗系统
- **8种武器类型**
- **13种敌人类型**
- **2个史诗Boss战**
  - 深海领主 (1000米)

- 流畅的战斗体验

## 🖥️ 系统需求

### 最低配置
- **操作系统**: Windows 10 64-bit / macOS 12+ / Ubuntu 20.04+
- **处理器**: Intel Core i3-6100 / AMD FX-6300
- **内存**: 4 GB RAM
- **显卡**: NVIDIA GTX 750 Ti / AMD R7 260X / Intel UHD 630
- **DirectX**: Version 11
- **存储**: 2 GB 可用空间
- **网络**: 宽带互联网连接 (联机模式需要)

### 推荐配置
- **操作系统**: Windows 10/11 64-bit / macOS 13+ / Ubuntu 22.04+
- **处理器**: Intel Core i5-8400 / AMD Ryzen 5 2600
- **内存**: 8 GB RAM
- **显卡**: NVIDIA GTX 1060 / AMD RX 580
- **DirectX**: Version 12
- **存储**: 2 GB 可用空间 (SSD推荐)
- **网络**: 宽带互联网连接

## 📦 安装指南

### Windows
1. 下载 `SebeJJ_v1.0.0_Windows.zip` 或 `SebeJJ_v1.0.0_Windows_Setup.exe`
2. 解压/运行安装程序
3. 运行 `SebeJJ.exe` 开始游戏

### macOS
1. 下载 `SebeJJ_v1.0.0_macOS.dmg`
2. 打开DMG文件，将应用拖入Applications文件夹
3. 从启动台运行游戏

### Linux
1. 下载 `SebeJJ_v1.0.0_Linux.tar.gz`
2. 解压: `tar -xzf SebeJJ_v1.0.0_Linux.tar.gz`
3. 运行 `./SebeJJ.x86_64` 开始游戏

## ⚠️ 已知问题

### 当前已知问题
1. **联机延迟**: 联机模式下偶尔出现延迟波动，重新连接房间可解决
2. **渲染问题**: 2000米区域在部分老旧显卡上可能出现渲染问题，建议降低画质设置
3. **排行榜同步**: 排行榜数据同步可能有短暂延迟
4. **隐藏成就**: 部分隐藏成就触发条件较为隐蔽

### 即将修复
以上问题计划在 v1.0.1 补丁中修复。
