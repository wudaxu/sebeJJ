# 赛博机甲 SebeJJ - Beta版本发布说明

**版本**: v0.2.0-Beta  
**发布日期**: 2026-06-11 (预计)  
**上一版本**: v0.1.0-Alpha  
**更新类型**: 重大更新

---

## 🎉 版本概述

欢迎进入 **赛博机甲 SebeJJ Beta测试阶段**！

Beta版本是游戏开发的重要里程碑，标志着游戏从功能完善阶段进入内容扩展与联机准备阶段。本版本带来了全新的深度区域、基础联机功能、完整的成就系统和排行榜，为正式发行奠定坚实基础。

---

## ✨ 主要新特性

### 🗺️ 全新深度区域

#### 1500米 - 深渊核心
探索前所未有的深渊领域，面对极限挑战：
- **全新环境**: 绝对黑暗、高压环境、地热喷口
- **5种新资源**: 深渊水晶、黑曜石矿脉、地热能量核心等传说级资源
- **5种新敌人**: 深渊吞噬者、暗影潜伏者、熔岩蟹等强力敌人
- **5个新委托**: 深渊探索、地热采集、吞噬者狩猎等挑战任务

#### 2000米 - 远古遗迹
揭开深海之下远古文明的秘密：
- **神秘场景**: 古代文明遗迹、神秘能量场
- **5种传说资源**: 远古遗物、能量矩阵、虚空金属等顶级资源
- **终极Boss战**: 远古守护者三阶段战斗，独特机制与丰厚奖励
- **遗迹解谜**: 探索古代机关，获取隐藏宝藏

### 🌐 多人联机（基础版）
与好友一起探索深海：
- **2-4人联机**: 支持小队协作探索
- **房间系统**: 创建/加入房间，快速匹配
- **好友邀请**: 邀请Steam好友一起游戏
- **协作机制**: 共享采集、协同作战、队友救援
- **状态同步**: 流畅的联机体验

### 🏆 成就系统
80个精心设计的成就等你解锁：
- **探索成就 (15个)**: 深度探索、区域解锁
- **战斗成就 (20个)**: 击败敌人、Boss击杀、连击挑战
- **收集成就 (15个)**: 资源收集、图鉴完成
- **机甲成就 (10个)**: 升级里程碑、装备收集
- **社交成就 (10个)**: 联机互动、好友添加
- **隐藏成就 (10个)**: 秘密挑战等你发现

成就等级：🥉铜级 🥈银级 🥇金级 💎白金

### 📊 排行榜系统
与全球玩家一较高下：
- **全球深度榜**: 最深下潜记录
- **全球财富榜**: 累计获得货币排名
- **全球击杀榜**: 击败敌人数量排名
- **全球速度榜**: 任务完成速度排名
- **好友排行榜**: 与好友比拼
- **周/月排行榜**: 定期重置，公平竞争

## 📋 系统要求

### 最低配置
- **操作系统**: Windows 10 64-bit
- **处理器**: Intel Core i3-6100 / AMD FX-6300
- **内存**: 4 GB RAM
- **显卡**: NVIDIA GTX 750 Ti / AMD R7 260X
- **DirectX**: Version 11
- **存储**: 2 GB 可用空间
- **网络**: 宽带互联网连接 (联机模式)

### 推荐配置
- **操作系统**: Windows 10/11 64-bit
- **处理器**: Intel Core i5-8400 / AMD Ryzen 5 2600
- **内存**: 8 GB RAM
- **显卡**: NVIDIA GTX 1060 / AMD RX 580
- **DirectX**: Version 12
- **存储**: 2 GB 可用空间 (SSD推荐)
- **网络**: 宽带互联网连接

## ⚠️ 已知问题

### Beta阶段已知问题
1. 联机模式下偶尔出现延迟波动
2. 2000米区域在部分显卡上可能出现渲染问题
3. 排行榜数据同步可能有短暂延迟
4. 部分隐藏成就触发条件过于隐蔽
