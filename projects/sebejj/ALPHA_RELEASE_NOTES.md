## 🚀 安装说明

### 系统要求

| 项目 | 最低配置 | 推荐配置 |
|------|----------|----------|
| 操作系统 | Windows 10 / macOS 12 / Ubuntu 20.04 | Windows 11 / macOS 14 |
| 处理器 | Intel i3 / AMD Ryzen 3 | Intel i5 / AMD Ryzen 5 |
| 内存 | 4 GB RAM | 8 GB RAM |
| 显卡 | Intel HD 630 | GTX 1050 / RX 560 |
| 存储空间 | 2 GB | 5 GB |
| Unity版本 | 2022.3 LTS | 2022.3 LTS |

### 安装步骤

1. **获取代码**
```bash
git clone <repository-url>
cd sebejj
```

2. **Unity Hub打开**
   - 启动 Unity Hub
   - 点击 "Open" → 选择 `sebejj` 文件夹
   - 确保使用 Unity 2022.3 LTS 版本

3. **首次导入**
   - 等待 Unity 完成资源导入
   - 打开 `Assets/Scenes/MainScene.unity`

4. **运行游戏**
   - 点击 Play 按钮开始游戏
   - 或选择 `File → Build Settings` 构建到目标平台

### 构建设置

```
File → Build Settings
├── Platform: PC, Mac & Linux Standalone
├── Target Platform: [Windows / macOS / Linux]
├── Architecture: x86_64
└── Compression Method: LZ4
```

## 🎮 操作指南

### 基础操作

| 按键 | 功能 | 说明 |
|------|------|------|
| `W/A/S/D` | 移动 | 控制机甲移动方向 |
| `Shift` | 推进器 | 临时加速，消耗能量 |
| `空格` | 扫描 | 扫描周围资源 |
| `Tab` | 切换目标 | 切换采集目标 |
| `E` | 采集/确认 | 采集资源或确认操作 |
| `ESC` | 菜单/取消 | 打开菜单或取消采集 |
| `I` | 背包 | 打开背包界面 |
| `F` | 攻击 | 使用当前武器 |
| `1/2/3` | 武器切换 | 切换武器槽位 |

### 游戏流程

1. **开始游戏** → 自动进入 Q001 新手试潜
2. **完成教程** → 学习基础操作
3. **接受委托** → 在基地委托板选择任务
4. **配置机甲** → 根据委托需求调整装备
5. **下潜探索** → 进入深海执行任务
6. **采集资源** → 扫描并采集目标资源
7. **遭遇战斗** → 使用武器击败敌人
8. **返回基地** → 完成任务获得奖励
9. **升级机甲** → 使用奖励强化装备
10. **继续冒险** → 接受新委托

## 🧪 测试参与

### 如何反馈

1. **Bug报告**: 使用 `docs/Alpha/BUG_REPORT_TEMPLATE.md` 模板
2. **功能建议**: 提交到项目管理工具

### 测试重点

1. **功能完整性**: 所有功能是否正常工作
2. **稳定性**: 是否出现崩溃或卡死
3. **性能**: 帧率是否稳定
