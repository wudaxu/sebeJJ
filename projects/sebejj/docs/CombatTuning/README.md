# 赛博机甲SebeJJ 战斗手感调优文档

## 概述
本文档记录了赛博机甲SebeJJ战斗系统的所有手感调优参数和设计理念。

## 设计原则

### 1. 响应性 (Responsiveness)
- 玩家输入到角色响应的时间应小于100ms
- 攻击前摇应让玩家有准备感但不拖沓
- 取消窗口允许玩家在攻击前摇期间取消动作

### 2. 反馈性 (Feedback)
- 每次命中都应有清晰的视觉、听觉、触觉反馈
- 屏幕震动、命中停顿、特效三位一体
- 伤害数字和受击动画强化命中感

### 3. 节奏感 (Rhythm)
- 攻击间隔形成可预测的节奏
- 连招系统提供流畅的攻击链
- 敌人生成节奏控制战斗强度

## 快速参考

### 关键参数范围

| 参数类型 | 推荐范围 | 说明 |
|---------|---------|------|
| 攻击前摇 | 0.1-0.3s | 近战较短，远程较长 |
| 攻击后摇 | 0.2-0.5s | 影响连击节奏 |
| 硬直时间 | 0.05-0.2s | 受击反馈强度 |
| 弹道速度 | 15-40 m/s | 远程武器 |
| 命中停顿 | 0.03-0.1s | 顿帧效果 |
| 屏幕震动 | 0.1-0.5强度 | 根据伤害调整 |
