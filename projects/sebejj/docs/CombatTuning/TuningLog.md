# 战斗手感调优记录

## 调优日志

### 2026-02-27 初始调优

#### 武器系统调优

**近战武器整体调整:**
- 问题: 原攻击前摇过长(0.3s)，感觉拖沓
- 调整: 缩短至0.15s，提升响应性
- 理由: 赛博机甲主题需要快节奏战斗

**光束军刀:**
- 前摇: 0.2s → 0.15s
- 后摇: 0.35s → 0.25s
- 硬直: 新增0.08s命中停顿
- 连招窗口: 0.8s
- 理由: 作为主力武器需要流畅手感

**等离子拳套:**
- 前摇: 0.1s → 0.08s (极快)
- 后摇: 0.2s → 0.15s
- 击退: 10 → 12 (强击退创造距离)
- 理由: 高风险高回报，快速打断

**震荡锤:**
- 前摇: 0.4s → 0.35s
- 后摇: 0.6s → 0.5s

**远程武器整体调整:**
- 弹道速度统一提升20%
- 命中判定半径增加50%

**脉冲步枪:**

**追踪导弹:**

## 调优原则总结

### 1. 响应性优先
- 攻击前摇控制在0.08-0.35s范围
- 快速武器<0.15s，重型武器<0.4s
- 取消窗口至少30%

### 2. 反馈明确
- 每次命中都有视觉+听觉+触觉反馈
- 停顿时间0.05-0.1s
- 震动强度与伤害成正比

### 3. 节奏流畅
- 连招窗口0.8s
- 敌人生成间隔15-30s
- 攻击后摇允许流畅衔接

### 4. 敌人差异化
- 快速型: 低硬直、高频率
- 防御型: 高减伤、反击机制
- 远程型: 预警明显、受击惩罚
