# SebeJJ 赛博机甲 - 游戏设计文档 (GDD)

## 1. 游戏概述

### 1.1 核心概念
《赛博机甲 SebeJJ》是一款**深海资源猎人**题材的2D动作探索游戏。玩家驾驶可高度自定义的赛博机甲，在危机四伏的深海中执行各种委托任务，采集珍稀资源，对抗机械怪物，逐步揭开深海之下的秘密。

### 1.2 游戏定位
- **核心体验**: 委托驱动 + 资源采集 + 机甲自定义
- **目标玩家**: 喜欢探索、收集和轻度Roguelike元素的玩家
- **游戏时长**: 单局15-30分钟，长期养成
- **平台**: PC (Steam) / 移动端

### 1.3 核心循环
```
┌─────────┐    ┌─────────┐    ┌─────────┐    ┌─────────┐
│  接受   │───►│  配置   │───►│  下潜   │───►│  探索   │
│  委托   │    │  机甲   │    │  深海   │    │  采集   │
└─────────┘    └─────────┘    └─────────┘    └────┬────┘
     ▲                                             │
     │    ┌─────────┐    ┌─────────┐    ┌─────────┘
     └────│  升级   │◄───│  返回   │◄───│  战斗   │
          │  机甲   │    │  基地   │    │  生存   │
          └─────────┘    └─────────┘    └─────────┘
```

## 2. 世界观设定

### 2.1 背景故事
公元2147年，地球表面资源枯竭，人类将目光投向深海。巨大的海底城市"深渊城"建立起来，成为人类最后的工业中心。

你是一名**自由机甲驾驶员**，驾驶着名为"SebeJJ"的赛博机甲，接受各种委托：采集稀有矿物、清除失控的机械生物、探索未知区域...在深海的压力和危险中，寻找生存和财富。

### 2.2 视觉风格
**赛博朋克 × 深海恐惧**

- **色调**: 深蓝、青绿、霓虹紫、警告橙
- **氛围**: 压抑、神秘、高科技与原始深海并存
- **参考**: 《Blame!》的机械美学 + 《深海迷航》的探索感

### 2.3 关键元素
- **机甲**: 模块化设计，可高度自定义
- **深海**: 分层设计，越深越危险，资源越珍贵
- **机械生物**: 失控的深海机械，既是威胁也是资源来源
- **深渊城**: 玩家的基地，升级和社交中心

## 3. 核心系统

### 3.1 委托系统

#### 委托类型
1. **采集委托** - 收集指定数量和类型的资源
2. **歼灭委托** - 击败特定敌人或数量
3. **探索委托** - 到达指定深度或发现新区域
4. **护送委托** - 保护NPC或货物到达目标点
5. **狩猎委托** - 击败Boss级敌人

#### 委托难度
- ★☆☆☆☆ 新手级 (0-30米)

### 3.2 机甲系统

#### 机甲框架 (3种)

**轻型框架 - "幽灵"**
- 高机动性，低耐久
- 适合：快速探索、躲避型玩法
- 基础属性：船体80，能量100，速度1.5，货舱50

**标准框架 - "守护者"**
- 平衡型
- 适合：新手、通用玩法
- 基础属性：船体120，能量100，速度1.0，货舱100

**重型框架 - "堡垒"**
- 高耐久，低机动
- 适合：战斗、重型采集
- 基础属性：船体180，能量80，速度0.7，货舱150

#### 装备系统 (8种类型)

| 装备类型 | 功能 | 稀有度 |
|----------|------|--------|
| 钻头 | 提升采集速度 | ★-★★★ |
| 机械爪 | 提升采集范围 | ★-★★★ |
| 武器 | 攻击力 | ★-★★★★★ |
| 护盾 | 防御/能量盾 | ★-★★★★ |
| 扫描器 | 探测范围 | ★-★★★ |
| 引擎 | 移动速度 | ★-★★★★ |
| 电池 | 能量上限 | ★-★★★ |
| 装甲 | 耐久上限 | ★-★★★★ |

#### 模块系统 (被动加成)

- **效率模块** - 采集速度+20%
- **强化模块** - 最大耐久+25%
- **超频模块** - 移动速度+15%
- **隐形模块** - 敌人侦测范围-30%
- **自修模块** - 缓慢自动修复
- **扩容模块** - 货舱容量+30%

## 4. 用户界面

### 4.1 界面结构

```
主菜单
├── 开始游戏
├── 继续游戏
├── 设置
└── 退出

基地场景
├── 委托板 (接受/查看委托)
├── 机甲工坊 (配置机甲)
├── 商店 (购买装备)
├── 仓库 (查看资源)
└── 下潜准备

下潜场景 (HUD)
├── 左上角: 耐久/能量
├── 右上角: 深度/氧气
├── 左下角: 小地图
├── 右下角: 武器/工具
└── 底部: 背包快捷栏
```

### 4.2 UI风格
- **主色调**: 深蓝 (#0a1628) + 青色 (#00d4aa)
- **警告色**: 橙色 (#ff6b35)
- **字体**: 等宽科技感字体
- **元素**: 全息投影风格，扫描线效果

## 5. 音效与音乐

### 5.1 背景音乐
- **基地**: 舒缓的电子音乐，营造安全感
- **浅海**: 神秘的氛围音乐
- **深海**: 压抑、紧张的低音
- **深渊**: 几乎寂静，偶尔的不明声响
- **Boss战**: 激烈的战斗音乐

### 5.2 音效
- 机甲移动: 机械液压声
- 采集: 钻探声 + 资源掉落声
- 战斗: 武器射击、爆炸、命中
- UI: 科幻感的点击、确认音
- 环境: 深海压力声、气泡声

## 6. 数值设计

### 6.1 经济系统
- 基础委托奖励: 500-1000信用点
- 中级委托奖励: 1500-3000信用点
- 高级委托奖励: 4000-10000信用点
- 资源价值: 10-500信用点/个

### 6.2 成长曲线
- 玩家等级: 1-50级
- 每级所需经验递增
- 等级解锁: 新装备、新委托、新区域

### 6.3 平衡目标
- 单次下潜收益: 1000-5000信用点
- 装备价格: 5000-50000信用点
- 机甲升级: 10000-100000信用点
