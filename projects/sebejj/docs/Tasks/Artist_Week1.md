# 美术组 Week 1 任务清单

## 概述
本周目标是确定美术风格，完成机甲基础设计。

## 任务详情

### 1. 美术风格确定 [P0]
**描述**: 确定赛博朋克深海风格的视觉方向
**验收标准**:
- [ ] 主色调色板确定
- [ ] 风格参考图收集整理
- [ ] 视觉风格文档
**预估工时**: 4小时
**风格关键词**: 赛博朋克、深海、机械、霓虹、压抑
**参考作品**: 
- Blame! (机械美学)
- 深海迷航 (深海氛围)
- 赛博朋克2077 (霓虹效果)

### 2. 机甲基础设计稿 [P0]
**描述**: 设计3种机甲框架的概念图
**验收标准**:
- [ ] 轻型框架"幽灵"概念设计
- [ ] 标准框架"守护者"概念设计
- [ ] 重型框架"堡垒"概念设计
- [ ] 设计说明文档
**预估工时**: 8小时
**设计要求**:
- 体现赛博朋克风格
- 模块化设计，便于装备展示
- 区分度明显（轻/中/重）

### 3. 机甲Sprite绘制 [P0]
**描述**: 完成第一个机甲框架的Sprite
**验收标准**:
- [ ] 标准框架"守护者"正面视图
- [ ] 标准框架"守护者"侧面视图
- [ ] 标准框架"守护者"背面视图
- [ ] 尺寸: 256x256像素
- [ ] 格式: PNG透明背景
**预估工时**: 12小时
**技术规格**:
- 分辨率: 256x256
- 色彩模式: RGB
- 文件格式: PNG (透明)
- 风格: 2D手绘/像素可选

### 4. 基础装备图标 [P1]
**描述**: 绘制8种基础装备的图标
**验收标准**:
- [ ] 钻头图标
- [ ] 机械爪图标
- [ ] 武器图标
- [ ] 护盾图标
- [ ] 扫描器图标
- [ ] 引擎图标
- [ ] 电池图标
- [ ] 装甲图标
- [ ] 尺寸: 64x64像素
**预估工时**: 8小时

### 5. UI框架设计 [P0]
**描述**: 设计游戏UI的整体风格
**验收标准**:
- [ ] 主菜单UI设计稿
- [ ] 游戏内HUD设计稿
- [ ] UI组件库（按钮、面板、进度条等）
- [ ] UI风格指南文档
**预估工时**: 8小时
**设计要求**:
- 全息投影风格
- 扫描线效果
- 科技感字体
- 主色: 深蓝+青色

## 本周交付物

- [ ] 3个机甲概念设计
- [ ] 标准机甲Sprite (3视图)
