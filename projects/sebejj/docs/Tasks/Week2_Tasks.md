# Week 2 任务清单

## 概述
Week 2 目标是实现委托系统和资源系统，完成核心游戏循环的基础部分。

**描述**: 搭建委托管理系统的核心架构
**验收标准**:
- [ ] MissionManager 单例实现
- [ ] 委托状态机 (Inactive/Active/Completed/Abandoned)
- [ ] 委托接受/放弃/完成功能
