## 五、录制指南

### 5.1 录制环境设置

#### Unity编辑器设置
```
1. 分辨率设置
   - Game窗口: 1920x1080 (16:9)
   - Scale: 1x (确保清晰度)

2. 画质设置
   - Quality: Ultra
   - 特效质量: High
   - 抗锯齿: 4x MSAA
   - 后处理: 开启

3. 录制工具
   - 推荐: OBS Studio / GeForce Experience
   - 编码: H.264
   - 比特率: 50 Mbps (录制用，后期压缩)
   - 帧率: 60fps
```

#### 游戏内设置
```
1. 显示设置
   - 显示UI: 开启
   - 显示伤害数字: 开启
   - 显示教程提示: 关闭 (避免干扰)

2. 调试功能
   - 无敌模式: 录制战斗画面时可开启
   - 快速采集: 录制采集画面时可开启
   - 时间控制: 用于捕捉特定时刻
```

### 5.2 分镜录制要点

#### 开场部分
- **镜头1.1**: 使用黑暗场景，仅开启探照灯
- **镜头1.2**: 使用自由相机环绕机甲
- **镜头1.3**: 使用场景相机展示深渊城全景

#### 核心玩法部分
- **镜头2.1**: 在浅海区录制，确保光线充足
- **镜头2.2**: 准备多个资源点，连续采集
- **镜头2.3**: 在空旷区域录制，避免遮挡
- **镜头2.4**: 使用无敌模式，确保流畅战斗
- **镜头2.5**: 准备可破坏岩石场景

#### 系统部分
- **镜头3.2**: 准备15个不同类型的委托
- **镜头3.3**: 确保商店有充足商品库存
- **镜头3.4**: 准备充足的升级材料
- **镜头3.5**: 三种框架都配置好装备

#### 深度探索部分
- **镜头4.1-4.4**: 在不同深度层录制，注意环境变化
- 使用深度传送功能快速切换

#### Boss战部分
- **镜头5.1-5.5**: 
  - 使用调试模式控制Boss血量
  - 分阶段录制，确保每个技能都展示
  - 准备多个角度镜头

### 5.3 后期制作建议

#### 剪辑要点
1. **节奏控制**: 战斗部分快节奏，系统部分慢节奏
2. **转场效果**: 使用闪白、推进、淡入淡出
3. **颜色校正**: 统一四个深度层的色调
4. **音效混合**: 游戏音效70%，背景音乐30%

#### 字幕规范
```
- 字体: 思源黑体 / Noto Sans CJK
- 主标题: 48px, 白色, 描边2px黑色
- 副标题: 32px, 青色(#00D4AA), 描边1px黑色
- 显示时长: 至少3秒，确保可读性
- 位置: 底部中央，距离底部10%
```

#### 音乐编辑
```
- 开场: 0:00-0:15 悬疑氛围音乐
- 核心玩法: 0:15-1:30 节奏感电子乐
- 系统展示: 1:30-3:00 舒缓科技音乐
- 深度探索: 3:00-4:00 紧张氛围音乐
- Boss战: 4:00-5:00 激烈战斗音乐
- 结尾: 5:00-5:20 主题音乐高潮
```
