```
驾驶可高度自定义的赛博机甲，潜入危机四伏的深海执行委托任务。采集珍稀资源，对抗机械怪物，在深渊的压力下生存并揭开海底的秘密。
```

### 详细描述 (About This Game)
```
公元2147年，地球表面资源枯竭，人类将目光投向深海。

在《赛博机甲 SebeJJ》中，你是一名自由机甲驾驶员，驾驶着可高度自定义的赛博机甲，在深渊城中接受各种委托任务。从采集稀有矿物到清除失控的机械生物，从探索未知区域到击败巨型Boss，每一次下潜都是一场生死考验。

【核心特色】

🤖 深度机甲自定义
• 3种机甲框架：轻型"幽灵"、标准"守护者"、重型"堡垒"
• 8种装备类型：钻头、机械爪、武器、护盾、扫描器、引擎、电池、装甲
• 6种功能模块：效率、强化、超频、隐形、自修、扩容
• 打造属于你的独特机甲配置

🌊 分层深海探索
• 4个深度层级：浅海(0-30m)、中层(30-60m)、深海(60-90m)、深渊(90-100m)
• 越深越危险，资源越珍贵
• 压强、能见度、敌人生成随深度变化

📋 委托驱动玩法
• 15个精心设计的委托任务
• 5种委托类型：采集、歼灭、探索、护送、狩猎
• 5个难度等级，从新手到深渊
• 完成终极试炼，成为传奇猎人

⚔️ 紧张刺激的战斗
• 对抗多种机械生物：机械鱼、机械蟹、机械水母
• 挑战巨型Boss"铁钳巨兽"
• 动能、能量、爆炸、腐蚀四种伤害类型

💎 丰富的资源系统
• 4个稀有度等级：普通、罕见、稀有、史诗
• 12种独特资源，从铜矿石到古代遗物
• 资源可出售或用于升级装备

【游戏特色】

• 单局15-30分钟，适合碎片时间游玩
• 长期养成系统，持续解锁新内容
• 赛博朋克×深海恐惧的独特视觉风格
• 沉浸式音效设计，感受深海的压迫感

准备好下潜了吗？深渊在等待。
```

## 📊 Steam页面优化建议

### 愿望单转化优化
1. **首图吸引力**: 胶囊图必须在0.5秒内传达游戏核心
2. **简短描述**: 前20字必须抓住眼球
3. **截图质量**: 前3张截图决定用户是否继续浏览
4. **视频前置**: 有视频时，前15秒决定留存率

### SEO优化
- 游戏名称包含关键词 "Cyber" "Mech"
- 描述中自然融入 "underwater exploration" "mecha customization"
- 标签选择精准，避免过于宽泛

- 设置 Steam 社区中心
- 定期发布开发者日志
- 及时回复玩家评论
- 收集反馈用于后续更新
