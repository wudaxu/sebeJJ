# 赛博机甲 SebeJJ - 测试计划文档

**版本**: v1.0

**测试负责人**: 测试团队

## 1. 项目概述

### 1.1 游戏信息
| 项目 | 内容 |
|------|------|
| 游戏名称 | 赛博机甲 SebeJJ |
| 游戏类型 | 深海资源猎人（委托驱动） |
| 美术风格 | 赛博朋克 |
| 技术栈 | Unity 2D |
| 目标平台 | PC (Windows/Mac/Linux) |

### 1.2 核心系统
1. **委托系统** - 接取、追踪、完成委托任务
2. **机甲系统** - 机甲控制（移动、扫描、采集）
3. **资源系统** - 氧气、能源、背包管理
4. **下潜系统** - 深度、压力、危险区域
5. **存档系统** - 游戏进度保存/加载

---

## 2. 测试目标

### 2.1 质量目标
- 功能完整性：所有核心功能正常工作
- 稳定性：崩溃率 < 0.1%
- 性能：目标帧率 60 FPS，最低接受 30 FPS
- 兼容性：支持 Windows 10/11, macOS 12+, Ubuntu 20.04+

### 2.2 测试范围
- **包含**: 所有游戏系统、UI、存档、性能
- **不包含**: 网络多人功能（如未来添加）

---

## 3. 测试策略

### 3.1 测试类型
| 测试类型 | 优先级 | 执行方式 |
|----------|--------|----------|
| 功能测试 | P0 | 手动 + 自动化 |
| 性能测试 | P0 | 自动化 |
| 兼容性测试 | P1 | 手动 |
| 回归测试 | P0 | 自动化 |
| 探索性测试 | P1 | 手动 |

### 3.2 测试阶段

#### 阶段一：单元测试（开发中持续进行）
- 各系统独立测试
- 自动化覆盖率目标：60%

#### 阶段二：集成测试（功能合并后）
- 系统间交互测试
- 委托流程端到端测试

#### 阶段三：系统测试（Alpha版本）
- 完整游戏流程测试
- 性能基准测试

#### 阶段四：验收测试（Beta版本）
- 15个初始委托完整验证
- 兼容性测试
